# norkyst-postpro
Post-processing tools for output files from the Norkyst ocean modelling system

Overview of tools:
- `about_norkystv3.ipynb`: Gives an introduction and overview of the ocean model and simple instructions on how to load and open data.
- `current_map.ipynb`: Example on how to plot with map projections and show currents.
- `current_rose.ipynb`: Example on how to plot a current rose for a given location.
- `ts_diagram.ipynb`: Example on how to produce a TS diagram for a given location and how to assess stability of the water column through density ratio / stability ratio.
- `timeseries.ipynb`: Examples on how to plot timeseries of Norkyst data.
- `cross_section.ipynb`: Example on how to plot and show a cross section, including transport through the section.
