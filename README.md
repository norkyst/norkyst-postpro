Post-processing tools for output files from the Norkyst ocean modelling system

Overview of tools:
- `about_norkyst.ipynb`: Gives an introduction and overview of the ocean model and simple instructions on how to load and open data.
- `current_map.ipynb`: Example on how to plot with map projections and show currents.
- `current_rose.ipynb`: Example on how to plot a current rose for a given location.
- `ts_diagram.ipynb`: Example on how to produce a TS diagram for a given location.
- `timeseries.ipynb`: Examples on how to plot simple timeseries of Norkyst outputs.
- `Python/Analysis`: Freshwater height, potential energy anomaly and Turner angle.
- `cross_section.ipynb` (coming): Example on how to plot and show a cross section, including transport through the section.
